> Beri salam ke mascot kotlin yaitu, Kodee!

- Kotlin lahir pada juli 2011 dan di kembangkan oleh [jetbrains](https://kotlinlang.org/) 
- Merupakan bahasa baru untuk JVM (*Java virtual machine*)
- Kotlin merupakan nama pulau di dekat St. petersburg Rusia hal ini mengikuti bahasa pendahulunya yaitu Java yang merupakan nama pulau Jawa di indonesia walau sebenarnya java lebih diartikan ke nama kopi.
- Kotlin versi 1.0 rilis 2016.
